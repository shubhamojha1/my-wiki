# RTX 4090

NVIDIA's Ada Lovelace architecture consumer flagship GPU. The best single-GPU option for deep learning practitioners as of 2023, offering near-data-center performance at consumer pricing.

## Specifications

| Spec | Value |
|------|-------|
| Architecture | Ada Lovelace |
| Tensor Cores | 4th gen ( Dedicated) |
| FP16 TFLOPS | ~330 (sparse) |
| FP8 TFLOPS | ~660 (0.66 PFLOPS) |
| Memory | 24 GB GDDR6X |
| Memory Bandwidth | ~1,008 GB/s |
| CUDA Cores | 16,384 |
| SMs | 128 |
| L2 Cache | 72 MB |
| TDP | 450W |
| Slots | 3-slot |
| Release | Oct 2022 |

## Key Features

### Fourth-Generation Tensor Cores
- Same Tensor Core generation as H100
- FP8 support (first consumer GPU with this)
- Improved sparse matrix support
- Full BF16 and TF32 support

### Massive L2 Cache
- **72 MB L2 cache** (12x more than Ampere's 6 MB)
- For BERT-Large: entire model fits in L2 cache during training
- Result: 1.5-2x speedup for this model class
- Larger models benefit less but still see gains

### FP8 Support
- 0.66 PFLOPS for FP8 matrix multiplication
- More compute than world's fastest supercomputer in 2007
- 4x RTX 4090 ≈ world's fastest supercomputer in 2010
- Mixed precision (keep some dims in FP16) required for stability

### AD102 Die
- Same architecture as professional RTX 6000 Ada
- No deliberate Tensor Core degradation (unlike RTX 30)
- Near-professional performance at consumer price

## Performance

### Raw Relative Performance (transformers)

| GPU | 16-bit Performance | 8-bit Performance |
|-----|------------------|------------------|
| H100 SXM | 1.0x | 1.0x |
| H100 PCIe | ~0.85x | ~0.85x |
| **RTX 4090** | ~0.4-0.5x | **~0.33x** |

### vs Previous Generations

| Comparison | Speedup |
|------------|---------|
| vs RTX 3090 | ~1.5-2x |
| vs RTX 3090 Ti | ~1.3-1.5x |
| vs A100 (16-bit) | ~0.4x (H100 is 3x A100) |

### Why 8-bit is Faster

FP8 has 2x faster data loading than FP16:
- Can load twice as much data per memory access
- More data fits in caches
- Combined with 4th gen Tensor Cores = massive speedup

## Practical Considerations

### Power and Cooling

**Power Draw**: 450W TDP — very power hungry
- Need quality 850W+ PSU for single GPU
- 4x RTX 4090 = 1800W just for GPUs
- Power limiting recommended for multi-GPU

**Power Connector Issues**:
- Uses 16-pin 12VHPWR connector
- Only 0.1% had melting issues (user error)
- Must insert cable until *click*
- Test by wiggling — should not move

**Cooling**:
- 3-slot design (won't fit 4 GPUs in standard case)
- Use PCIe extenders for multi-GPU
- Throttles at 83°C
- Blows hot air into case — airflow critical

### Size and Compatibility

- **Length**: ~336mm (14.4 inches)
- **Slots**: 3-slot (won't fit in many small cases)
- **Weight**: Heavy — need case support

### Multi-GPU Setup

**Challenges**:
- No NVLink (consumer cards)
- PCIe bandwidth is bottleneck for multi-GPU
- 4x setup needs 1600W+ PSU
- Cooling difficult in standard cases

**Solutions**:
- PCIe extenders for cooling and spacing
- Power limiting (reduce 50W = only 7% slower)
- Alternative: Use single RTX 4090 for most tasks

## Who Should Buy RTX 4090?

### Best For:
- Individual researchers and practitioners
- Running models up to ~20B parameters
- Training medium-sized models
- Fine-tuning large models (with gradient checkpointing)
- 8-bit inference for LLMs

### Consider Alternatives For:
- **Multi-GPU training**: A100/H100 (NVLink support)
- **Very large models**: H100 (80GB vs 24GB)
- **Maximum memory**: A100/H100 80GB
- **Enterprise**: Data center GPUs

## Comparison to RTX 3090

| Feature | RTX 4090 | RTX 3090 |
|---------|----------|----------|
| Architecture | Ada | Ampere |
| Tensor Cores | 4th gen | 3rd gen |
| L2 Cache | 72 MB | 6 MB |
| FP8 | ✓ | ✗ |
| VRAM | 24 GB | 24 GB |
| Memory BW | 1 TB/s | 936 GB/s |
| TDP | 450W | 350W |
| Price | ~$1,600 | ~$1,500 (original) |

**Verdict**: RTX 4090 is ~50% faster for similar price. The L2 cache alone justifies upgrade.
