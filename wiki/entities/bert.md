# BERT

**BERT** (Bidirectional Encoder Representations from Transformers) is a transformer-based language model that uses bidirectional encoder architecture.

## Key Features

- **Bidirectional**: Reads text left-to-right AND right-to-left
- **Masked Language Modeling**: Predict masked tokens during pretraining
- **Next Sentence Prediction**: (Removed in later versions)
- **Pre-train + Finetune**: Large pretraining, task-specific finetuning

## Impact

- Released by Google in 2018
- Sparked the pretraining era in NLP
- GLUE benchmark introduced

## Variants

- **RoBERTa**: Optimized BERT training
- **ALBERT**: Parameter-efficient BERT
- **DistilBERT**: Knowledge distilled version
