# RTX 3090

NVIDIA's Ampere architecture consumer flagship GPU (2020). Popular for deep learning due to 24GB VRAM and good Tensor Core performance at consumer pricing.

## Specifications

| Spec | Value |
|------|-------|
| Architecture | Ampere |
| Tensor Cores | 3rd gen |
| FP16 TFLOPS | ~356 (sparse) |
| Memory | 24 GB GDDR6X |
| Memory Bandwidth | 936 GB/s |
| CUDA Cores | 10,496 |
| SMs | 82 |
| L2 Cache | 6 MB |
| TDP | 350W |
| Slots | 2-slot |
| Release | Sep 2020 |

## Key Features

### Third-Generation Tensor Cores
- Full support for TF32, BF16, FP16, INT8
- Structured sparsity support (2x throughput on sparse models)
- Same Tensor Core generation as A100

### 24GB VRAM
- Large for consumer GPU at launch
- Can fit most models without offloading
- Comparable to workstation Quadro cards

### Ampere Architecture
- Improved IPC over Turing
- Better ray tracing (gaming)
- More efficient than RTX 2080 Ti

### vs Previous Generation (RTX 2080 Ti)

| Metric | Improvement |
|--------|-------------|
| Tensor Core perf | ~2x |
| Memory | 24GB vs 11GB |
| Memory BW | 936 vs 616 GB/s |

### Sparse Model Performance

With structured sparsity (sparse weights):
- Effective throughput can double
- Requires sparse training algorithms
- Not automatic — need to train sparse models

## Tensor Core Degradation (RTX 30 Series)

**Important**: Unlike Quadro/Titan cards, NVIDIA deliberately reduced Tensor Core utilization in RTX 30 series consumer GPUs.

**What this means**:
- A100 Tensor Cores are ~10-15% more efficient
- Not a dealbreaker, but factor into cost/performance
- RTX Titan (上一代) had no degradation

**For comparison**:
- RTX 3090 vs RTX Titan: ~10-15% slower per Tensor Core
- RTX 3090 vs A100: ~10-15% slower per Tensor Core

## Practical Considerations

### Power and Cooling

**Power Draw**: 350W TDP
- Single GPU: 650W+ PSU recommended
- 4x RTX 3090: 1600W+ PSU needed
- Can use power limiting to reduce heat

**Cooling**:
- 2-slot design (better than RTX 4090)
- Found in both blower and triple-fan designs
- Blower better for multi-GPU (exhausts outside case)
- Triple-fan better for single GPU (quieter, cooler)

### Multi-GPU Setup

**Challenges**:
- 4x RTX 3090 = 1400W GPU power
- No NVLink on consumer cards
- Cooling 4 GPUs difficult

**Solutions**:
- PCIe extenders for spacing and cooling
- Use blower-style cards for multi-GPU
- Consider 2x RTX 3090 instead of 4x

### Power Limiting for Multi-GPU

**Technique**: Reduce power limit to fit in PSU budget

| Power Limit | Performance Impact |
|-------------|-------------------|
| Stock (350W) | Baseline |
| -50W (300W) | ~7% slower |
| -100W (250W) | ~15% slower |

**Recommendation**: Reduce by 50W for multi-GPU stability.

## Who Should Still Buy RTX 3090?

### Good Reasons:
- Budget-conscious (used market ~$600-800)
- Need 24GB but can't afford RTX 4090
- Multi-GPU setup (2x cheaper than 2x RTX 4090)
- Running older models that don't need latest features

### Better Alternatives:
- RTX 4090 for new builds
- Used RTX 3090 Ti for slightly more performance
- A100 80GB for large models
