# OSI Layer 7: Application

**Layer 7** of the OSI model is the layer users interact with directly, handling high-level application protocols and user-facing actions.

## Responsibility

- User-facing protocols and actions
- Application-level functions like authentication
- High-level network services

## Key Details

- Hosts all user-facing protocols
- Most visible to end users
- Note: Modern protocols often consolidate Session (L5) and Presentation (L6) into this layer

## Examples

- HTTP/HTTPS (web requests)
- DNS (domain name resolution)
- SMTP/IMAP (email)
- FTP/SFTP (file transfer)
- SSH (remote access)
- DHCP (IP address assignment)
