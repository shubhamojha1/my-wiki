# SHA-256

**SHA-256** is part of the SHA-2 family, producing 256-bit output.

## Characteristics

- **Output**: 256 bits (64 hex characters)
- **Secure**: No known practical collision attacks
- **Standard**: Current default for file integrity

## Use Cases

- File downloads verification
- Software integrity
- Digital signatures
- Blockchain
